## 📌 About The Project

This project helps users to track their expenses categorized by date, category, and amount. It provides monthly summaries and insights through a secure, responsive, and modern web application.

### Features:
- 🔐 User authentication (JWT)
- ➕ Add/Edit/Delete expenses
- 🔍 Filter by category and date range
- 📊 Monthly expense summary
- 💬 Toast notifications for feedback
- 📱 Fully responsive UI

## 🚀 Getting Started

### ✅ Prerequisites

- Node.js (v16 or higher)
- npm (v8 or higher) or yarn
- MongoDB Atlas account or local MongoDB instance
- Code editor (VS Code recommended)

---

### 🛠️ Installation

1. **Clone the repository**
```bash
git clone https://github.com/jazimahmed/ExpenseTracker.git
cd ExpenseTracker
```

---
### 🛠️ set up backend

1. **Clone the repository**
   
```bash
cd backend
npm install
nodemon server.js
```

---
### 🛠️ set up .env file
3. **Create a .env file in the backend directory**
```bash
PORT=5000
MONGO_URI=your_mongodb_connection_uri
JWT_SECRET=your_jwt_secret_key
```

---
### 🛠️ Frontend setup
4. **Navigate to the frontend folder:**
```bash
cd ../frontend
npm install
npm run dev
```

## 📂 Usage

- Register a user account.
- Login to your dashboard.
- Add expenses, filter by date or category.
- View summary by month.
